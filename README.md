# Bachelor Thesis

## Frequentist and Bayesian Logistic Regression for Heart Disease Prediction

This repository contains the R code developed for my Bachelor's thesis at the University of Groningen.

The project compares frequentist and Bayesian logistic regression approaches for heart disease prediction using:

- Multiple Imputation by Chained Equations (MICE)
- Frequentist logistic regression with backward selection
- Bayesian logistic regression using Hamiltonian Monte Carlo (HMC)
- Bayesian variable selection using spike-and-slab priors
- Cross-validation
- Subsampling-based robustness analysis

The main scripts are located in the `R-Code` folder.

Detailed instructions on the execution order of the scripts, required inputs, generated outputs, and adaptation to new datasets can be found in:
